These tests cover mathematical functions.

### Trigonometric

```
>>> let pi = acos(-1.0)

>>> sin([0.0, pi / 4., pi / 2., 3. * pi / 4. , pi])
[0.0, 0.707107, 1.0, 0.707107, 0.0]

>>> cos([0.0, pi / 4., pi / 2., 3. * pi / 4. , pi])
[1.0, 0.707107, 0.0, -0.707107, -1.0]

>>> tan([0.0, pi / 4., 3. * pi / 4. , pi])
[0.0, 1.0, -1.0, -0.0]

>>> asin([-1., 0., 1.])
[-1.570796, 0.0, 1.570796]

>>> acos([-1., 0., 1.])
[3.141593, 1.570796, 0.0]

>>> atan([-1., 0., 1.])
[-0.785398, 0.0, 0.785398]

```
